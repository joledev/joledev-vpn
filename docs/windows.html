Configurar WireGuard en Windows

Descargar WireGuard

Descarga el instalador oficial desde wireguard.com/install. Ejecuta el .msi e instala.

Obtener tu configuracion

Descarga el archivo .conf desde el panel admin.

Importar el tunel

Abre WireGuard, haz clic en "Importar tunel(es) desde archivo" y selecciona tu archivo .conf.

Activar

Selecciona el tunel y haz clic en "Activar". El estado cambiara a "Activo" con estadisticas de transferencia.

Verificar que funciona

Abre el navegador y ve a:

https://whatismyip.com

La IP debe ser la IP de tu VPS.

Tambien puedes verificar desde PowerShell:

(Invoke-WebRequest -Uri "https://ifconfig.me").Content

Solucion de problemas

Error "acceso denegado" WireGuard necesita permisos de administrador. Haz clic derecho en la app y selecciona "Ejecutar como administrador".
No se conecta Verifica que el Firewall de Windows no bloquee WireGuard. Agrega una excepcion para wireguard.exe en Firewall de Windows Defender.
Conflicto con otro VPN Desactiva cualquier otro VPN antes de conectar WireGuard. No pueden coexistir dos VPNs full-tunnel.
Internet lento Edita el tunel en WireGuard y cambia el MTU a 1420 en la seccion [Interface].

Nota de seguridad

Que protege Cifra todo tu trafico. Nadie en la red WiFi puede ver lo que haces.
Que NO protege No reemplaza un antivirus. No oculta tu identidad de sitios web que ya te tienen identificado. El .conf tiene tu llave privada; no lo compartas.
