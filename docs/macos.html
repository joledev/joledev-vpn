Configurar WireGuard en macOS

Instalar WireGuard

Descarga WireGuard desde la Mac App Store o con Homebrew:

brew install wireguard-tools

La app de la App Store incluye interfaz grafica. Homebrew solo instala las herramientas CLI.

Obtener tu configuracion

Descarga el archivo .conf desde el panel admin o pide el QR al administrador.

Importar la configuracion

App Store: Abre WireGuard > "Importar tunel(es) desde archivo" > selecciona el .conf.

CLI:

sudo install -m 600 laptop-extra.conf /etc/wireguard/wg0.conf
sudo wg-quick up wg0

Conectar

Activa el tunel desde la app (icono en la barra de menus) o con:

sudo wg-quick up wg0

Verificar que funciona

curl -s ifconfig.me

Debe devolver la IP de tu VPS.

Solucion de problemas

Permisos de macOS macOS pedira permiso para agregar una configuracion VPN en Preferencias del Sistema. Acepta y autenticate.
Homebrew: wg-quick no encontrado Asegurate de que /opt/homebrew/bin este en tu PATH. Ejecuta: export PATH="/opt/homebrew/bin:$PATH".
Firewall de macOS bloqueando Ve a Preferencias del Sistema > Red > Firewall y asegurate de que no bloquee conexiones salientes UDP.

Nota de seguridad

Que protege Todo tu trafico sale cifrado por el VPS. Nadie en tu red local puede interceptarlo.
Que NO protege No te hace anonimo. No sustituye un antivirus. El archivo .conf contiene tu llave privada; no lo compartas.
