Configurar WireGuard en iOS

Instalar WireGuard

Descarga WireGuard desde la App Store. App oficial y gratuita.

Obtener tu configuracion

El administrador te proporcionara un codigo QR o un archivo .conf.

Importar la configuracion

Opcion A (QR): Abre WireGuard, toca + > "Crear desde codigo QR". Escanea el QR.

Opcion B (archivo): Si recibes un archivo .conf por AirDrop o mensajeria, abrelo y selecciona "Abrir en WireGuard".

Permitir configuracion VPN

iOS pedira permiso para agregar una configuracion VPN. Toca "Permitir" y autenticate con Face ID / Touch ID.

Conectar

Activa el interruptor del tunel. Veras el icono "VPN" en la barra de estado del iPhone.

Verificar que funciona

Con el VPN activado, visita:

https://whatismyip.com

Debe mostrar la IP de tu VPS.

Solucion de problemas

No se conecta Verifica tu conexion a Internet. Algunas redes WiFi corporativas bloquean UDP en puertos no estandar.
Se desconecta al bloquear la pantalla En WireGuard, edita el tunel y activa "On-Demand" para que se reconecte automaticamente.
DNS no resuelve Edita el tunel y verifica que el campo DNS tenga: 1.1.1.1, 1.0.0.1.

Nota de seguridad

Que protege este VPN Cifra todo tu trafico entre tu iPhone y el VPS. Ideal para redes WiFi publicas.
Que NO protege No te hace anonimo. No protege contra apps maliciosas ni phishing. El VPS tiene una IP fija.
